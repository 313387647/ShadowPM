# ShadowPM 外测快速上手

这份文档可以直接发给外部测试者。

## 你要测试什么

ShadowPM 会尝试把一份项目 Excel 变成一个项目管控工作区：

- `管控总表`：项目事项、负责人、部门、截止日期、状态、进度结论。
- `资金账本`：预算确定、拆分、合并、增补、划拨、支出、退回等预算流转记录。
- `执行日历`：日期、渠道、负责人、内容、状态。
- `项目活动`：每次管控事项、预算、日历的变更记录。

重点不是页面好不好看，而是 AI 生成的结构是否比原表格更容易管理。

## 测试入口

1. 打开测试链接：`https://<测试域名>/demo`
2. 点击 `下载案例表格`，或准备你自己的项目 Excel。
3. 点击 `开始测试`。
4. 在登录页选择 `林小夏`。

如果你是管理者或评审者，可以之后选择 `陈鹏` 查看反馈后台。

## 路线 A：上传 Excel 生成项目

1. 进入 `AI 工作台`。
2. 点击 `新建项目`。
3. 选择 `AI 生成`。
4. 上传 Excel。
5. 点击 `开始解析`。
6. 等待 AI 生成预览。
7. 检查项目名称、时间、管控表、预算流水、执行日历。
8. 点击 `创建项目与管控表`。

如果预算缺失或待确认，也可以先创建。ShadowPM 的目标是先生成可编辑管控表，再让人快速补齐。

## 路线 B：手动创建项目

1. 回到 `AI 工作台`。
2. 点击 `新建项目`。
3. 切换到 `手动创建`。
4. 填写项目名称、日期和可选预算。
5. 创建后进入项目详情页。
6. 在 `管控总表` 新增并修改管控事项。

这条路线用来测试：不依赖 AI 时，ShadowPM 是否仍然能作为轻便项目管控工具使用。

## 检查重点

请重点观察：

- AI 是否把预算估算误当成真实支出。
- AI 是否把预算写进管控总表，而不是拆到资金账本。
- AI 是否把负责人、渠道、内容混在一起。
- 缺失字段是否能直接在管控表里补齐。
- 手动新增管控事项是否容易找到。
- 每次状态、进度、预算或日历修改是否留下活动记录。
- 整体流程是否点击太多，或者让你不敢继续。

## 提交反馈

进入项目详情页后，点击上方 `外测反馈` 展开表单。

请填写：

- 综合评分
- AI 识别质量
- 上传结果
- 后续使用意愿
- 主要问题勾选
- 哪一步最卡
- 其他反馈

好的反馈示例：

```text
AI 把“预计预算 20 万”当成已经支出的费用。
```

```text
执行日历里负责人和渠道混在一起，我不知道该怎么改。
```

```text
手动创建项目后能进入详情页，但新增管控事项入口不够明显。
```

## 完成标准

一次有效测试至少包含：

- 登录成功
- 完成一次 AI 上传创建，或一次手动创建
- 进入项目详情页
- 检查并修改管控表
- 检查资金账本或执行日历
- 提交外测反馈
